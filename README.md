AI Stock Assistant 1.0

A Python-based intelligent stock analysis tool for beginners.

Features
Multi-timeframe analysis
Trend detection (MA)
Risk analysis (volatility)
RSI indicator
Smart advice system
BUY / HOLD / AVOID signals
How to Run
pip install -r requirements.txt
python stock_assistant.py
Example Usage
Enter stock: TCS  
Choose timeframe: 1y  

Output:
Trend: Mixed  
Risk: Medium  
Signal: HOLD
